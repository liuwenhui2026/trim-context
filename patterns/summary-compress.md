# Summary Compress: 旧消息摘要替代原文

> 🚧 Work in Progress

把早期对话用 LLM 生成摘要，替代原始消息放入上下文。

## 思路

对话超过阈值时，启动子 Agent 对旧消息做摘要，用一段摘要文本替换数百条原始消息。

## 适用场景

- 长会话，历史信息需要保留语义而非原文
- 对话主题逐渐演进，早期细节不重要

## 优缺点

- ✅ 保留语义，不像滑窗直接丢弃
- ❌ 摘要本身消耗 Token 和延迟
- ❌ 摘要可能丢失关键细节
- ❌ 摘要质量依赖模型能力

## 框架实现

- Claude Code: Autocompact（4级压缩最后一环）
- Hermes: Autocompact
